Interfaces

IDefaultStatusBarItemsProvider

A specialized provider responsible for managing the default or core status bar items that appear when the application initializes.

IStatusBarItem

Represents an individual visual element or status indicator to be displayed in the status bar.

IStatusBarItemsProvider

Manages the registration and retrieval of status bar items, allowing for modular contribution to the application's bottom bar.

IStatusBarManager

The central coordinator that manages all status bar contributions. It aggregates items from various providers and categorizes them by their target sections.

IStatusBarView

Defines the visual contract for the status bar view component. Provides methods to access runtime layout properties.

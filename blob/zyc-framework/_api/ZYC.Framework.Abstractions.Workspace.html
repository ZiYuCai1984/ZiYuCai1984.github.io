Namespace ZYC.Framework.Abstractions.Workspace

Classes

CustomWorkspaceLayout

Represents a user-defined workspace layout preset, including its metadata, preview thumbnail, and the workspace tree used to restore it.

MoveToNewWorkspaceTabItemHeaderContextMenuSubItem

Represents a specific context menu item used to move a tab into a newly created workspace. Unlike its base class, this item does not target an existing node but rather triggers the creation of a new one.

MoveWorkspaceTabItemHeaderContextMenuSubItem

Represents a specific sub-item in the context menu used for moving workspace tabs. It links a target WorkspaceNode with the command required to perform the move.

WorkspaceNode

Represents a node within a hierarchical workspace layout tree. Can be either a parent node (defining a split) or a leaf node (containing navigation state).

WorkspaceNodeEx

Provides extension methods for the WorkspaceNode class to facilitate tree traversal, searching, indexing, and debugging of the workspace layout.

Interfaces

IMoveWorkspaceTabItemHeaderContextMenuItemManager

Defines a manager responsible for generating sub-items for the "Move Tab" context menu. This is typically used to provide a list of target workspace nodes where a tab can be relocated.

IParallelWorkspaceManager

Defines the contract for managing parallel workspaces, handling layout operations such as splitting, merging, and workspace state management.

IWorkspaceContextMenuManager

Defines a specialized manager for workspace-related context menu items. Inherits from IMenuManager<TItem> to provide standard lifecycle and collection management for IWorkspaceMenuItem objects.

IWorkspaceMenuItem

Defines the contract for a menu item within the workspace UI, such as context menus or navigation bars.

IWorkspaceMenuManager

Defines a specialized manager for workspace-related menu items. Inherits from IMenuManager<TItem> to provide standard lifecycle and collection management for IWorkspaceMenuItem objects.

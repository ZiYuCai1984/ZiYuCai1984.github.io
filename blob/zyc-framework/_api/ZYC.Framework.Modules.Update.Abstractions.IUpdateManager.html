Provides high-level operations for managing product updates, including fetching update metadata, downloading the update payload, and applying the update.

public interface IUpdateManager

Remarks

Implementations are expected to maintain and update an UpdateContext that reflects the latest known update state. Methods returning UpdateContext should return the updated context after the operation completes.
